# React Map 專案

這是一個使用 React + TypeScript + Vite 開發的地圖應用，展示台北市 YouBike 站點資訊。

## 功能特點

- 互動式地圖顯示 YouBike 站點位置
- 站點資訊即時更新
- 站點使用量統計圖表
- 站點搜尋功能
- 響應式設計

## 技術棧

- React 18
- TypeScript
- Vite
- Leaflet (地圖庫)
- React Leaflet
- Recharts (圖表庫)

## 本地開發

1. 克隆專案
```bash
git clone https://github.com/james-wu-1101/react-map.git
cd react-map
```

2. 安裝依賴
```bash
npm install
```

3. 啟動開發服務器
```bash
npm run dev
```

4. 構建生產版本
```bash
npm run build
```
